# 🎯 START HERE - New User Guide

## What is This?

A **Python trading bot simulator** for Binance Futures. It's completely safe - no real trading happens!

---

## ⚡ Run in 30 Seconds

**Step 1:** Open terminal in the `trading_bot` folder

**Step 2:** Run this command:
```bash
python cli.py
```

**Step 3:** Enter when prompted:
- Symbol: `BTCUSDT`
- Side: `BUY`
- Type: `MARKET`
- Quantity: `0.01`

Done! You just simulated your first order. ✅

## 🎮 Two Ways to Use

### 1. Interactive Mode (Easier)
```bash
python cli.py
```
The program asks you questions step-by-step.

### 2. CLI Mode (Faster)
```bash
python cli.py --symbol BTCUSDT --side BUY --type MARKET --quantity 0.01
```
Everything in one command.
